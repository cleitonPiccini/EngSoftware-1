Para o funcionamento do Software desenvolvido na disciplina de 
Engenharia de Software é necessario os seguintes programas:

* Servidor Apache 
	- Xampp 32 bits. = https://www.apachefriends.org/pt_br/download.html
	- O Xampp ja possue o SGBD e os recursos para rodar o Software.

* Apos intalar o servidor Apache
	- Abrir o XAMPP Control Panel e dar start nos seguintes recursos.
		- Apache
		- MySQL

* Apos start dos recursos:
	- Abrir o navegador e acessar a seguinte URL = http://localhost/phpmyadmin/server_import.php
	- Escolher o arquivo contendo a base de dados. "\trabalho\sql\posto_marcon.sql"
	- Executar.

* Adicionar a pasta "trabalho" para o servidor.
	- Copiar a pasta trabalho para - \xampp\htdocs\

* Abrir o navegador:
	- Acessar a seguinte URL = http://localhost/trabalho/
